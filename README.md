# 千田和央 ビジネスサイト

トップページのみの静的Webサイトです。`index.html` をブラウザで開くと確認できます。

## 公開前に変更する箇所

- `script.js` の `contact@example.com` を実際の問い合わせ先メールアドレスに変更してください。
- 必要に応じて `index.html` のフッター年や文言を調整してください。

## ファイル構成

- `index.html`: ページ本体
- `styles.css`: デザイン
- `script.js`: 問い合わせフォームのメール起動処理
- `assets/hero-organization.png`: ヒーロー用ビジュアル
